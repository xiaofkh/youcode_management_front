# YouCode Management 前端项目

## 技术栈

| 类别 | 技术 | 版本 | 说明 |
|------|------|------|------|
| 框架 | React | 18.x | UI 框架 |
| 语言 | TypeScript | 5.x | 类型安全 |
| 构建工具 | Vite | 6.x | 快速开发和构建 |
| 样式 | Tailwind CSS | 4.x | 原子化 CSS |
| UI 组件 | Radix UI | - | 无样式可访问组件 |
| 状态管理 | Jotai | 2.x | 原子化状态管理 |
| 服务端状态 | React Query | 5.x | API 数据缓存和管理 |
| HTTP 客户端 | Axios | 1.x | API 请求 |
| 路由 | React Router | 7.x | 页面路由 |
| 图标 | Lucide React | - | 图标库 |
| 工具库 | date-fns | 4.x | 日期处理 |
| 数据校验 | Zod | 3.x | 运行时类型校验 |

## 环境要求

- Node.js >= 18.x
- pnpm >= 8.x (推荐) 或 npm >= 9.x

## 项目启动

### 1. 安装依赖

```bash
cd youcode-management-front
pnpm install
# 或
npm install
```

### 2. 开发模式

```bash
pnpm dev
# 或
npm run dev
```

访问 http://localhost:3000

### 3. 生产构建

```bash
pnpm build
# 或
npm run build
```

构建产物在 `dist` 目录

### 4. 预览构建结果

```bash
pnpm preview
# 或
npm run preview
```

## API 代理配置

开发环境下，所有 `/api` 开头的请求会被代理到 `http://localhost:8000`。

如需修改后端地址，编辑 `vite.config.ts`:

```typescript
server: {
  proxy: {
    "/api": {
      target: "http://your-backend-url",
      changeOrigin: true,
    },
  },
}
```
